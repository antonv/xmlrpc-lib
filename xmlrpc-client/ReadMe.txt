========================================================================
ПРИЛОЖЕНИЕ WIN32. Обзор проекта xmlrpc-client
========================================================================

Это приложение xmlrpc-client создано автоматически с помощью 
мастера приложений. 

   В этом файле представлена сводка содержимого всех файлов, входящих в
   состав приложения xmlrpc-client.


xmlrpc-client.vcproj
   Основной файл проекта VC++, автоматически создаваемый с помощью 
   мастера приложений. 
   В файле представлены сведения о версии Visual C++, используемой для 
   создания файла, а также сведения о параметрах платформы, конфигурации 
   и проекта, установленных с помощью мастера приложений.

xmlrpc-client.cpp
   Основной исходный файл приложения.

/////////////////////////////////////////////////////////////////////////////
C помощью мастера приложений автоматически создаются следующие ресурсы:

xmlrpc-client.rc
   Перечень всех ресурсов Microsoft Windows используемых в программе при 
   компиляции для платформ, использующих стандартную оболочку Windows CE. К 
   ним относятся значки, растровые изображения и курсоры, хранящиеся в 
   подкаталоге RES. Этот файл можно редактировать непосредственно в Microsoft 
   Visual C++. При сохранении файла .rc определения в разделе данных 
   сохраняются в виде шестнадцатеричного представления числовых значений 
   (понятное имя определения в этом случае не используется).

xmlrpcclient.rc2
   В этом файле содержатся ресурсы, которые не могут редактироваться в среде 
   Microsoft Visual C++. В этот файл следует помещать все ресурсы, которые не 
   могут редактироваться с помощью редактора ресурсов.

Resource.h
   Стандартный файл заголовков, в котором определяются новые идентификаторы 
   ресурсов.
   В Microsoft Visual C++ осуществляется чтение и обновление содержимого 
   этого файла.

xmlrpc-clientppc.rc
   Перечень всех ресурсов Microsoft Windows, используемых в программе при 
   компиляции для платформы Pocket PC 2003 или платформ, поддерживающих 
   аналогичную модель пользовательского интерфейса. К ним относятся значки, 
   растровые изображения и курсоры, хранящиеся в подкаталоге RES. Этот файл 
   можно редактировать непосредственно в Microsoft Visual C++. При сохранении 
   файла .rc определения в разделе данных сохраняются в виде шестнадцатеричного 
   представления числовых значений (понятное имя определения в этом случае не 
   используется).

xmlrpcclientppc.rc2
   В этом файле содержатся ресурсы, которые не могут редактироваться в среде 
   Microsoft Visual C++. В этот файл следует помещать все ресурсы, которые не 
   могут редактироваться с помощью редактора ресурсов.

Resourceppc.h
   Стандартный файл заголовков, в котором определяются новые идентификаторы 
   ресурсов.
   В Microsoft Visual C++ осуществляется чтение и обновление содержимого этого 
   файла.




xmlrpc-client.ico
   Файл значка, используемого в качестве значка приложения (32x32).
   Этот значок включается в основной файл ресурсов xmlrpc-client.rc.

/////////////////////////////////////////////////////////////////////////////
Другие стандартные файлы:

StdAfx.h, StdAfx.cpp
   Эти файлы используются для построения файла предкомпилируемых заголовков 
   (PCH) xmlrpc-client.pch и файла предкомпилируемых типов StdAfx.obj.
